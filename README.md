# 🥊 Muay Thai Club Lorquí

Sitio web oficial del **Muay Thai Club Lorquí**, ubicado en Lorquí, Murcia, España.

## 🇹🇭 Acerca de Nosotros
   
Somos un club dedicado a enseñar el arte marcial tailandés auténtico con técnicas profesionales, disciplina y respeto. Ofrecemos clases para todas las edades y niveles de experiencia.

## 📋 Características

- ✨ **Diseño Moderno**: Interfaz elegante y responsiva con Claude Design
- 🎯 **Múltiples Clases**: Mini Fighters, Jóvenes, Adultos Principiantes, Adultos Avanzados, Fitness y Privadas
- 📍 **Ubicación Clara**: Mapa interactivo y información de contacto
- 📱 **Mobile Friendly**: Completamente responsivo en todos los dispositivos
- 🌐 **Formulario de Contacto**: Conexión directa con el club
- 🇹🇭 **Temática Tailandesa**: Elementos visuales de Tailandia integrados

## 🚀 Instalación Local

1. Clona el repositorio:
```bash
git clone https://github.com/tu-usuario/muay-thai-club.git
```

2. Abre `index.html` en tu navegador o utiliza un servidor local:
```bash
python -m http.server 8000
# o
npx http-server
```

3. Accede a `http://localhost:8000` en tu navegador
